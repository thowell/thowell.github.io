Achtung, Die Kurve!

Since reading about DeepMind's work on using deep reinforcment learning to play Atari games, I've wanted to explore these approaches myself. In this project, I worked with ALP to train a reinforcement learning agent using open-source tools to play the the cult-classic computer game Achtung, Die Kurve!. Ultimately, our experiments training the agent have not been successful yet, but we developed our own version of the game in Python based on existing implementations. You can download and play our version of the game with up to 3 friends!
