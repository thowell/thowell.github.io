Ping Pong Ball Shooting Robot

This project was my first foray into mechatronics and programming. I wrote code that analyzed an image, using MATLAB, to determine the location of the target holes. This analysis was the input to Arduino C code I also wrote to determine and control where to position the launcher, at what angle to fire, how hard to shoot, and the reloading sequence. The hardware was provided as a kit with instructions for assembly. My partner on the project was key to dialing in the system once I had completed the code. During the competition we made 5/6 holes which was good enough to receive an automatic A on the final!
